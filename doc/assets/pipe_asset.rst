Pipe asset
+++++++++++++++++++++++++++++++++++++++++++++
The **Pipe Asset** represents a pipe within the simulator. Pipes are used to transport fluids between assets.
This asset is stored within the `pipe` class in the `omotes_simulator_core.entities.assets.pipe` module.
